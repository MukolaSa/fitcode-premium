Мені не вистачає часу. Чи підійде?
+
Саме для тебе. FitCode побудований на мікро-тренуваннях по 20 хвилин. Займатись можна вдома, в готелі, у парку. Єдина умова — 20 хвилин на день.
Я абсолютний новачок. Підійде мені?
+
Так! В модулі 2 є спеціальний блок «з нуля». Дмитро сам починав без спортивного минулого і знає всі підводні камені для новачків.
Скільки часу буде доступ до матеріалів?
+
Безстроково. Матеріали завжди доступні у твоєму кабінеті. Доступ до закритого чату — теж назавжди.
Чи потрібне спеціальне обладнання?
+
Ні. Всі тренування для домашніх умов без інвентарю. Якщо є килимок — чудово. Якщо ні — теж ок.
Коли онлайн-сесія з Дмитром?
+
Наприкінці курсу — приблизно через 4 тижні після старту. Точна дата у закритому чаті. Всі учасники отримають запис.
